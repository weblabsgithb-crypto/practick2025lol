Аннотация проекта

Проект направлен на создание учебного прототипа централизованной системы сбора, визуализации и аналитики операционных показателей образовательной организации с добавлением простой предикативной модели для прогноза ключевых метрик.

Цель — cоздать автоматизированную отчётность, которая с помощью прогнозирование помогают принимать обоснованные управленческие решения в вузе. В результате был разработан прототип веб‑интерфейса, набор ETL‑скриптов для агрегации тестовых данных и базовая модель прогнозирования.
